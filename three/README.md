pytest对用例的要求:
1.文件：test_*.py / *_test.py
2.用例函数：
函数名 test_ 开头，非下划线私有函数
允许携带 fixture / 参数化参数（无参只是特例）
3.测试类场景：
类名 Test 开头，无 __init__
类内方法 test_ 开头，带 self
4.可调用实体：仅顶层函数 / 测试类方法（普通对象、lambda、内置函数不行）
5.无强制限制：返回值、函数参数（支持传参）
6.执行要求：断言使用原生 assert；异步用例依赖插件
